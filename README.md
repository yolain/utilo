## utilo

[![npm](https://img.shields.io/npm/v/utilo.svg)](https://www.npmjs.com/package/utilo) 


前端业务开发中多个项目会频繁使用到可复用的工具类函数，为了避免不同项目中因变动而重复拷贝，参考并整合一个可实用的工具库。

### 安装
``` bash
$ npm i utilo -S
```
